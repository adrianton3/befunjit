befunjit

Changelog
---------

####v0.2.0 --- 22 jul 2014
 + funge-space is now toroidal

####v0.1.0 --- 21 jul 2014

 + supports `^<v>`, `0-9`, `+-*/%`, `|_`, ``!` ``, `&.` and `p`
